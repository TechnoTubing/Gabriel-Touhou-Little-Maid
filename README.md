I made this because I like the idea of this mod (https://www.curseforge.com/minecraft/mc-mods/touhou-little-maid) but I wanted a cooler companion. 

_"I larp, therefore I am."_

How to use:
- Download the zip (or use `git clone https://github.com/TechnoTubing/Gabriel-Touhou-Little-Maid`)
- Find your instance folder
- Place it in "tlm-custom-models"
- Play the game!
- I haven't set it to automatically switch the sound but I will soon, for now you have to do it separately.

Yes, the icon sucks but yk...
